# AmenTauhid.github.io

Personal portfolio website themed as an Apple iMessage conversation. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no build step.

**Live:** [amentauhid.github.io](https://amentauhid.github.io)

## How It Works

The portfolio is presented as an interactive iMessage chat. Visitors pick quick-reply options, stage them in the input bar, and hit send to reveal sections of the portfolio through a conversational flow.

### Features

- **Interactive conversation engine** — pick topics via quick-reply buttons, stage in input, send to reveal content
- **Dark/light mode** — auto-detects OS preference, manual toggle with persistence
- **iMessage send effect** — bubble slam animation when messages are sent
- **Typing indicators** — animated dots while "Ayman" responds
- **iMessage reactions** — click any received bubble to react with emoji (heart, thumbs up, laugh, etc.)
- **Pinned topic navigation** — circular progress indicators below the header
- **Emoji status** — header emoji changes per section being viewed
- **Read receipts** — update from "Delivered" to "Read [time]" after viewing
- **Sound effects** — optional send/receive sounds with mute toggle
- **Case study modals** — click any project card for a detailed iOS-style modal
- **Real timestamps** — messages use the actual current time
- **Responsive design** — optimized for desktop, tablet, and mobile
- **Konami code** — try it for a surprise

## Sections

- **Projects** — 7 project cards with detailed case study modals and GitHub links
- **Work Experience** — 5 roles with expandable bullet points
- **Skills** — categorized tech stack grid
- **About** — career journey and mission statement
- **Contact** — GitHub and LinkedIn links

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Inter)
- Hosted on GitHub Pages
